# Building background images

Drop one WebP per building here, named `<slug>.webp` (e.g. `smithy.webp`, `main_building.webp`).
Optional **tribe-specific** plates are `<tribe>_<slug>.webp` (`romans_`/`teutons_`/`gauls_`, e.g.
`teutons_smithy.webp`): a building page shows its village's tribe plate when present, else the neutral
`<slug>.webp`, else just the dark theme (graceful, no broken-image artifact).
Prompts + the full slug list: `docs/art/building-backgrounds.md`.
