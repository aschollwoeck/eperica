# Unit portrait images

Drop one WebP per unit here, named `<tribe>_<id>.webp` (e.g. `romans_legionnaire.webp`,
`teutons_ram.webp`, `gauls_settler.webp`); wild animals use the bare id (`wolf.webp`).
Prompts + the full slug list: `docs/art/building-backgrounds.md` (Troop & unit art section).
Missing files fall back gracefully — the roster shows a dark placeholder tile, no broken-image icon.
