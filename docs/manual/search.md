# Finding players, alliances & places

The **search box** in the top bar (and the **Search** page) gets you anywhere fast.

## What you can search

- **Players** — type the start of a username to find players; each result links to their profile.
- **Alliances** — type the start of an alliance **name** or **tag** to find alliances; each result links to
  the alliance's stats.
- **Coordinates** — type a tile like `12|-7` (also `(12|-7)`, `12,-7`, or `12 -7`) and you'll get a direct
  link to that spot on the map.

Matching is by the **beginning** of the name or tag, and it's not case-sensitive. An empty search just
prompts you to type; a search with nothing matching says so plainly.

Search only shows public information — names, tags, and coordinates — and is open to everyone, even before
you log in.
